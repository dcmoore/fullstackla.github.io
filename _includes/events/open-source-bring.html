Don't forget to bring:

  • A laptop (with keyboard/mouse of your choice) 💻 🎹
  • A project you want to improve. (optional)
  • The most important thing: an open mind 💭💭💭

You should really preinstall or get:

  • Atom ⚛: We all love our very special development environments, but pair programing is all about collaboration. It really sucks when Vim gets in the way.

  • Ruby: At Fullstack LA, we love Ruby and you or may not need it for your open source projects. But it's nice to know that you already have it preinstalled if you work on, say, Homebrew.

  • Homebrew 🍺: Since we use Ruby, you'll eventually come across Homebrew when you start working on future open source projects. Might as well preinstall it just in case.
  • npm and Node.js: One of our current porjects, Hood.ie, is a JavaScript based open source project that relies on npm. Just install Node.js (which comes installed with npm) and you'll have both on your machine for later use.

  • A GitHub account: We're very active on GitHub and without it, you won't have access to our meetup show notes highlighting the evening's salient topics or the daily musings/issues we post on our GitHub repo.
