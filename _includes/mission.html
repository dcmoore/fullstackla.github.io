Mission

Cultivating Open Community & Open Software.

Fullstack is a group of software developers focoused on better code and better coders. We're deeply committed to open source. Everything we do is open and open to community improvement. We love well crafted open software. Events organize and drive these goals.

We value:

Mentorship
Guiding and counseling the next generation of fullstack developers.
Open Source
Committing to free and open source code that quickly helps people build things.
Inclusive Community
Welcoming any and all that want to learn.
Skill Building
Teaching new web development skills while refining the older essential skills.
